Install dependencies
====================

If you haven't installed [Typescript 2](https://www.typescriptlang.org) yet:

    npm install -g typescript@2

Install local dependencies

    npm install .

Build documentation
===================

Install `typedoc`

    npm install -g typedoc

Run `typedoc`

    typedoc --out docs src
